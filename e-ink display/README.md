# The Display

The display is a Waveshare 2.7" parasitic-NFC-powered e-ink display. It uses technology similar to wireless charging to power itself when in range of your phone and updates the screen to the data that is shared to it via NFC. The whole writing process takes a few seconds, and there are no batteries to charge. The display I bought use Version 2, Revision 2.2. This is apparently the latest version which has new chips that make the transfer faster, but it means the SDK that Waveshare provides does not work. They do have an app (Android and iOS) that allows you to write content to the screen, either by typing text or by using an image from your phone, but it is not the most intuitive. I had to write my own implementation to communicate with the display and transfer data
